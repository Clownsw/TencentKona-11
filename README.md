# Tencent JDK 11
![Tencent-kona.jpg](/uploads/53E12E7E40644E75B522447E91862F81/Tencent-kona.jpg)
- 免费、定制优化的OpenJDK版本
 - 基于OpenJDK11，兼容Java SE标准
 - 经过数据平台部多个系统内测使用，保障稳定性
 - 经SPECjvm、SPECjbb测试，性能比肩Oracle、Corretto、Azul等主流JDK11版本
 - 季度安全更新
